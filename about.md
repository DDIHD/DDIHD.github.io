# About Minimal Mistakes

A flexible two-column Jekyll theme perfect for building personal sites, blogs, and portfolios. Built with these awesome features:

![Team Photo]({{ site.baseurl }}/assets/img/team.png)



## Features

- Compatible with GitHub Pages
- Responsive layout
- Customizable navigation
- Optional header images
- Social media integration
- SEO best practices
- Clean, efficient code

## Support

Need help? Check out our [documentation](https://github.com/pages-themes/minimal) or [contact support](https://github.com/contact) and we'll help you sort it out.

## Contributing

Interested in contributing to Minimal Mistakes? We'd love your help. The theme is open source, and you can find it on [GitHub](https://github.com/pages-themes/minimal).
